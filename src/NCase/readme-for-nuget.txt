Congratulation, you have succesfully installed *NCase*
=========================================================

NCase is a mix between a Mocking Framework like Moq and a parametrized test framework, having advanced combinatorial capabilities.

NCase is not released yet! The API is now quite stable, but further commits may introduce breaking changes. Please give as much feedback as possible, positive, negative, critics!

For an introduction and examples, look at: 

    http://github.com/jeromerg/NCase
