Congratulation, you have succesfully installed *NDocUtil*
=========================================================

NDocUtil is an utility class used to generate the snippets in the NCase Documentation: 

    http://github.com/jeromerg/NCase

It generates C# code and Console snippets to use in your documentation.

During the build process, you can automatically:

- Export the snippets as text file
- Export the colored snippets as 
	- HTML file 
	- PNG image
	- Lossless WMF/EMF image
- Update markdown documentation 
- Update Powerpoint code snippets
