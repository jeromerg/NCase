Congratulation, you have succesfully installed *NUtil*
=========================================================

NUtil contains Various C# Utility classes, that were initially developed for the NCase project.


    http://github.com/jeromerg/NCase


Current list of utilities:

- Pairwise Generator
- ForEach
- Quadratic Processing
    - CartesianProduct
	- TriangularProductWithoutDiagonal
- Processing of Chained Dictionaries
	- CascadeAdd
	- CascadeRemove
	- CascadeGetOrDefault
	- CascadeTryFirst
- String Processing
	- CascadeAdd
	- Lines and JoinLines
	- Desindent and Indent
